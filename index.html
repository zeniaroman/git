Get git version git --version git version 2.15.0

Clone This command clonind remote repo to local workspace in folder with name 'git' git clone git@github.com:zeniaroman/git.git Cloning into 'git'...

Pull This command pull last version of repo from remote git pull Already up to date.

Check status Will show repo stage status git status On branch...

Repo log Show log of last changes. Press Q to exit git log commit 94f5db46c35c77111c8adeb4fc7219a1cdc73f4e (HEAD -> main, origin/main, origin/HEAD) Author: zeniaroman <45962172+zeniaroman@users.noreply.github.com> Date: Fri Feb 4 22:35:36 2022 +0300 Initial commit

Repo stages Add to index staging all modified files git add . (nothing) Commit changes to local repo git commit -m "Message for new changes" [main 6331f74] Message for new changes Reset the last commit that was not pushed and save commit changes to the workspace. It's --soft reset by default git reset HEAD~1 (nothing) Reset the last commit that was not pushed and don't save commit changes(any code) to the workspace. git reset --hard HEAD~1 (nothing) Push changes to remote repo git push Counting objects: ..., done. Revert changes on remote repo with clean commit code. Grab commit hash with git log git revert <commit_hash> :wq (in Vim) git push Counting objects: ..., done.

Work with branch Check current branch git branch * main Create new branch and switch to a this new branch git checkout -b <new_branch_name> Switched to a new branch 'new_branch_name' Create new branch without switch to a this new branch git branch <new_branch_name> (nothing) Switch between branch git checkout <branch_name> Switched to branch 'branch_name' Delete a local branch from repo.
